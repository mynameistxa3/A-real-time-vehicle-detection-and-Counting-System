# A Real-Time Vehicle Detection and Counting System

## Overview

This project presents a real-time vehicle detection and counting system using deep learning and computer vision techniques. The system detects vehicles from a live camera or video stream, tracks their movement, and counts vehicles crossing a predefined virtual line.

## Features

- Real-time vehicle detection
- Vehicle tracking
- Automatic vehicle counting
- Support for video files and webcam
- Export counting results

## Technologies

- Python
- OpenCV
- Ultralytics
  - YOLOv8
  - YOLO11
- NumPy
- Ultralytics
